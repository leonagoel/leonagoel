*React.js · Node.js · Express · MongoDB · JWT · Cloudinary*

*Python · FastAPI · Scikit-learn · Supabase · NLP*

**Key Features:** Cold-start NLP engine · FastAPI REST endpoints · Real-time Supabase (PostgreSQL) data layer
